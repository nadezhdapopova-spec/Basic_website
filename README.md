# Проект "Basic_website"

## Описание:

"Basic_website" является учебным проектом по Python-разработке. 

Проект содержит код для тестирования работы простого veb-приложения.
В проекте содержатся HTML-документы, являющиеся прототипами veb-страниц, с подключенными к ним Bootstrap-стилями.

В проекте используется ООП-подход для создания простого локального веб-сервера,
обработки входящих GET-запросов и POST-запросов.

## Установка:

1. Клонируйте репозиторий проекта:
````
git clone https://github.com/nadezhdapopova-spec/Basic_website.git
````
2. Установите зависимости внутри каталога проекта для создания виртуального окружения и установки зависимостей:
````
poetry install
````

## Использование:
На любой GET-запрос возвращается страница «Контакты».
Результатом POST-запроса является печать в консоль всех данных, которые были приняты от пользователя.

Проверка GET-запросов и POST-запросов может осуществляться с помощью приложения Postman.
